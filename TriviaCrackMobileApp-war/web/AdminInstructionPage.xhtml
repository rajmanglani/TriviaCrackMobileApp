Admin Instructions

Adding Questions

Adding questions to the question bank is very simple. Just follow these simple steps.

For adding multiple choice questions

1. Provide a unique ID for the question

2. Enter the Question

3. Select the Question Level

4. Select the Question Category

5. Enter the Correct answer

6. Enter 3 mandatory options and you may add 3 other optional choices

7. Click on the Create Question button to store the question in the question bank

For adding Fill in the blank questions

1. Click on the 'Add fill in the blank question' button

2. Provide a unique ID for the question

3. Enter the question, pick a level and a category

4. Enter the correct answer

5. Click on the 'Create Question' Button

Adding T/F questions

1. Click on the 'Add T/F question' button

2. Provide a unique ID for the question

3. Enter the question, pick a level and a category

4. Choose the correct answer by expanding the drop down (T/F)

5. Click the 'Create Question' Button

Precautions

1. While adding multiple choice questions, if the correct answer is all of the choices you provided, name that option "All are Correct"

2. While adding a multiple choice question, if the correct answer is a combination of two of the choices you have provided, refrain from adding any optional choices. Also refrain from using terms such as "Both A and B". Use terms that describe both options again.

Removing Questions

Simply click on the remove questions button; this will direct you to a page with a list of all the questions and a button to remove a question.
